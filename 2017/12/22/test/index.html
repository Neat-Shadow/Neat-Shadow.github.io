利用github搭建个人博客：hexo

hexo之本地搭建

一、git、Node.js、Hexo的安装

####1.Git
去官网下载https://git-scm.com/。

####2.Node.js
去官网https://nodejs.org/en/点击Download即可。

####3.Hexo
1)在桌面右键: Git Bash Here，利用npm命令安装。
npm install -g hexo
2)在本地创建名为hexo的文件夹，例如 D:/hexo，然后进入hexo文件夹右键git bash，输入
hexo init
自动生成建站需要的文件
3)继续输入
npm intall
自动安装依赖
4)生成静态文件
hexo generate
5)最后输入
hexo server
启动hexo本地服务，打开 http://localhost:4000/ 就能看到搭建好的本地博客了。
